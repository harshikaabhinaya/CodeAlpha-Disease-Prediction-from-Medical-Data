# Disease Prediction from Medical Data

This project develops a machine learning-based Disease Prediction System to predict the likelihood of diseases using patient medical data. It analyzes features such as age, symptoms, blood test results, and other clinical parameters to classify whether a patient is at risk of a particular disease. The project includes data preprocessing, exploratory data analysis, model training, performance evaluation, and prediction using classification algorithms such as Logistic Regression, Random Forest, SVM, and XGBoost. The objective is to support early disease detection, improve diagnostic accuracy, and assist healthcare professionals in making informed decisions.
